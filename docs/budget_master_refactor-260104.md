# 예산 관리 시스템 정규화 설계 (초안)

## 1. 설계 목적

본 문서는 교회 예산 관리 시스템에서  
- 조직 구조(위원회 / 사역팀)
- 예산 분류 체계(항 / 목 / 세목)
- 연도별 예산 관리
- 예산 승인 및 예외 결재 규칙  

을 **정규화된 데이터 모델**로 정리하기 위한 설계 초안이다.

---

## 2. 요구사항 요약

### 2.1 조직 및 담당자
- 각 위원회, 사역팀(부)에는 팀장이 존재
- 사역팀 팀장이 예산의 1차 승인자 역할 수행

### 2.2 예산 관리 기준
- 예산은 다음 기준으로 관리
  - 연도
  - 위원회
  - 사역팀(부)
  - 예산(항)
  - 예산(목)
  - 예산(세목)
- 실제 예산 금액은 **세목 단위**로 관리

### 2.3 승인 예외 규칙
- 기본: 사역팀 팀장이 1차 승인
- 예외: **공간사역팀 + 인건비(항)** → 재정팀장 전결
