# 신규 지출결의서 양식 분석

## 개요
기존 지출결의서(Ver.4.1.3)와 신규 양식(Ver.4.1.4)의 차이점 분석

---

## 주요 차이점 요약

| 구분 | 기존 (Ver.4.1.3) | 신규 (Ver.4.1.4) |
|------|-----------------|-----------------|
| 예산항목 선택 | 상단에서 1개 카테고리 선택 | **"하단 예산항목 참조" - 각 항목별 선택** |
| 세부항목 구조 | 예산(세목), 적요, 단가, 수량, 금액 | **예산(항/목/세목), 적요, 단가, 수량, 금액** |
| 위원회/사역팀 | 필수 선택 | **없음** |
| 청구인 | 1명 | 1명 (동일) |
| 은행 정보 | 은행명, 계좌번호, 예금주 | 동일 |
| 금액 계산 | 자동 (단가 × 수량, 10원 절사) | 동일 |

**핵심 차이점**: 예산항목을 헤더에서 한 번 선택하는 것이 아니라, **각 세부항목마다 개별 선택**

---

## 상세 비교

### 1. 헤더 영역

#### 기존 양식
```
┌─────────────────────────────────────────────────────────┐
│ 로고 │ 지출결의서 │ 예산항목: [위원회 선택] / [사역팀]   │
│      │           │ 지출일자: 2025년 __ 월 __ 일        │
│      │           │ 청구금액: ₩ XXX,XXX 원              │
└─────────────────────────────────────────────────────────┘
```
- 예산(항), 예산(목) 표시
- 위원회, 사역팀 선택 필요

#### 신규 양식
```
┌─────────────────────────────────────────────────────────┐
│ 로고 │ 지출결의서 │ 예산항목: "하단 예산항목 참조"      │
│      │           │ 지출일자: 2025년 __ 월 __ 일        │
│      │           │ 청구금액: ₩ XXX,XXX 원              │
└─────────────────────────────────────────────────────────┘
```
- **예산항목은 테이블에서 각 항목별로 선택**
- **위원회, 사역팀 선택 없음**

---

### 2. 세부 항목 테이블

#### 기존 양식 (5열)
| 예산(세목) | 적요 | 단가 | 수량 | 금액 |
|-----------|------|------|------|------|
| 교육자료비 | 세미나 자료 | 10,000 | 5 | 50,000 |

- 예산(세목)만 선택 (상위 카테고리는 헤더에서 이미 선택)
- 단가 × 수량 = 금액 (10원 단위 절사)

#### 신규 양식 (7열)
| 예산(항) | 예산(목) | 예산(세목) | 적요 | 단가 | 수량 | 금액 |
|----------|----------|-----------|------|------|------|------|
| 교역자사례비 | 교역자식대 | 교역자식대 | 11월분 식대 | 200,000 | 1 | 200,000 |
| 사무사역비 | 사무간사식대 | 사무간사식대 | 11월분 식대 | 200,000 | 1 | 200,000 |

- **각 항목별로 예산(항), 예산(목), 예산(세목) 선택**
- **여러 예산 카테고리를 하나의 결의서에 포함 가능**
- 단가 × 수량 = 금액 (10원 단위 절사) - **기존과 동일**

---

### 3. 청구인 정보 (동일)

```
○ 청구인: [이름] (인)
```
- 청구인 1명

---

### 4. 은행 정보 (동일)

```
○ [은행명]  ○ 계좌번호: [XXX-XXX-XXX]  ○ 예금주: [이름]
```
- 은행명, 계좌번호, 예금주 직접 입력
- 저장된 계좌 선택 기능 사용 가능

## 데이터 모델 변경 필요사항

### 신규 Expense 모델 (제안)

```prisma
model SimpleExpense {
  id              String   @id @default(cuid())

  // 지출일자
  expenseDate     DateTime?

  // 청구금액 (자동 계산)
  requestAmount   Int

  // 세부 항목들
  items           SimpleExpenseItem[]

  // 청구 정보
  requestDate     DateTime
  applicantName   String    // 청구인 (1명)

  // 은행 정보 (기존과 동일)
  bankName        String
  accountNumber   String
  accountHolder   String

  // 메타
  version         String   @default("4.1.4")
  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt
}

model SimpleExpenseItem {
  id              String   @id @default(cuid())
  expenseId       String
  expense         SimpleExpense @relation(fields: [expenseId], references: [id], onDelete: Cascade)

  // 예산 항목 (각 행마다 선택)
  budgetCategory     String   // 예산(항)
  budgetSubcategory  String   // 예산(목)
  budgetDetail       String   // 예산(세목)

  // 적요
  description     String   // 적요

  // 금액 (자동 계산)
  unitPrice       Int      // 단가
  quantity        Int      // 수량
  amount          Int      // 금액 (단가 × 수량, 10원 절사)

  order           Int
}
```

## UI 변경 필요사항

### 1. 새 메뉴 추가
- `/expenses/simple/new` - 신규 간편 지출결의서 작성
- `/expenses/simple` - 간편 지출결의서 목록
- `/expenses/simple/[id]` - 간편 지출결의서 상세

### 2. 폼 컴포넌트
- `SimpleExpenseForm.tsx` - 신규 양식 전용 폼
- **위원회/사역팀 선택 제거**
- **각 항목별 예산(항/목/세목) 드롭다운**
- 청구인 1명 (기존과 동일)
- 은행 정보 (기존과 동일, BankAccountSelector 재사용)
- 금액 자동 계산 (기존과 동일)

### 3. 프린트 컴포넌트
- `SimplePrintHeader.tsx` - "하단 예산항목 참조" 표시
- `SimplePrintBody.tsx` - 7열 테이블 구조 (항/목/세목/적요/단가/수량/금액)
- `SimplePrintFooter.tsx` - 청구인, 은행정보 (기존과 유사)

## 구현 우선순위

1. **Phase 1: DB 스키마**
   - SimpleExpense, SimpleExpenseItem 모델 추가

2. **Phase 2: API**
   - `/api/simple-expenses` CRUD API

3. **Phase 3: 폼 UI**
   - SimpleExpenseForm 컴포넌트
   - 예산항목 행별 선택 로직 (ItemBudgetSelector)

4. **Phase 4: 목록/상세 페이지**
   - 목록, 상세, 수정 페이지

5. **Phase 5: 프린트**
   - 신규 양식 프린트 레이아웃

## 참고: 예산항목 구조

신규 양식에서 확인된 예산항목 예시:
- 교역자사례비 > 교역자식대 > 교역자식대
- 사무사역비 > 사무간사식대 > 사무간사식대
- 목회활동비 > 목회_통신비 > 목회_통신비

기존 BudgetMaster 테이블의 category/subcategory/detail 구조와 동일하게 사용 가능

## 기존 양식과의 차이점 정리

| 항목 | 기존 (Ver.4.1.3) | 신규 (Ver.4.1.4) | 비고 |
|------|-----------------|-----------------|------|
| 예산항목 | 헤더에서 1개 선택 | 각 항목별 선택 | **주요 차이점** |
| 위원회/사역팀 | 필수 | 없음 | **제거** |
| 청구인 | 1명 | 1명 | 동일 |
| 은행정보 | 입력 | 입력 | 동일 |
| 금액계산 | 자동 | 자동 | 동일 |
| 테이블 열 | 5열 | 7열 | 항/목 추가 |
