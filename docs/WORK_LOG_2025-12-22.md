## 2. 사용자 정보 구조 변경 (userid/username 분리)

### 변경 내용
기존 `username` 필드를 `userid`와 `username`으로 분리했습니다.

| 필드 | 용도 | 예시 |
|------|------|------|
| userid | 로그인/결재 비교용 | 청연정혜종 |
| username | 화면 표시용 | 정혜종 |

### 수정된 파일 (9개)
- `lib/users.ts` - UserInfo 인터페이스에 userid 추가
- `app/api/auth/login/route.ts` - userid로 인증
- `app/api/auth/me/route.ts` - 응답에 userid 포함
- `app/login/page.tsx` - userid로 로그인
- `components/Header.tsx` - username 표시
- `components/ExpenseForm.tsx` - 청구인에 userid 자동입력
- `components/SimpleExpenseForm.tsx` - 청구인에 userid 자동입력
- `app/expenses/[id]/page.tsx` - 결재 비교에 userid 사용
