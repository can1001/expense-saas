### 4. budget-view 페이지 - 동일한 사용금액 계산 적용

**커밋:** `7c4114c` feat: budget-view 페이지 사용금액도 지급완료 지출결의서 합계로 계산

**수정 파일:**
- `app/api/budget/hierarchy/route.ts`

동일한 로직 적용:
```typescript
const usedAmounts = await prisma.expenseItem.groupBy({
  by: ['budgetDetail'],
  where: {
    expense: {
      paymentStatus: 'COMPLETED',
      expenseDate: { gte: new Date(year, 0, 1), lt: new Date(year + 1, 0, 1) },
    },
  },
  _sum: { amount: true },
});
```
