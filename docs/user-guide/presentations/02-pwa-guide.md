# PWA 사용 가이드

**지출결의서 앱 설치 및 오프라인 사용**

---

## PWA란?

### Progressive Web App

웹 기술로 만든 앱으로, 네이티브 앱과 유사한 경험 제공

### 장점

| 장점 | 설명 |
|------|------|
| 설치 가능 | 홈 화면에 아이콘으로 추가 |
| 오프라인 지원 | 인터넷 없이도 사용 가능 |
| 빠른 로딩 | 리소스 캐싱으로 빠른 시작 |
| 자동 업데이트 | 앱스토어 없이 자동 업데이트 |
| 저용량 | 네이티브 앱 대비 적은 용량 |

---

## 앱 설치 - iOS (iPhone, iPad)

### 설치 순서

1. **Safari** 브라우저로 사이트 접속
2. 하단의 **공유 버튼** (□↑) 탭
3. 스크롤하여 **"홈 화면에 추가"** 선택
4. 이름 확인 후 **"추가"** 탭

### 주의사항

> ⚠️ iOS에서는 **Safari**에서만 PWA 설치 가능
> Chrome, Firefox에서는 설치 옵션이 나타나지 않습니다

---

## 앱 설치 - Android

### Chrome에서 설치

1. Chrome으로 사이트 접속
2. 주소창 옆 **메뉴 (⋮)** 탭
3. **"앱 설치"** 또는 **"홈 화면에 추가"** 선택
4. **"설치"** 확인

### 자동 설치 배너

사이트를 여러 번 방문하면 화면 하단에 설치 배너가 자동으로 표시됩니다

**"설치"** 버튼 탭 → 바로 설치!

---

## 앱 설치 - Desktop (PC, Mac)

### Chrome

1. 사이트 접속
2. 주소창 오른쪽의 **설치 아이콘** (⊕) 클릭
3. **"설치"** 확인

### Edge

1. 사이트 접속
2. 주소창 오른쪽의 **앱 설치 아이콘** 클릭
3. **"설치"** 확인

---

## 설치 후 사용

### 앱 실행
- 홈 화면(모바일) 또는 앱 목록(데스크톱)에서 **"지출결의서"** 아이콘 탭/클릭
- 브라우저 주소창 없이 전체 화면으로 실행

### 바로가기 기능 (앱 아이콘 길게 누르기)

| 바로가기 | 이동 화면 |
|----------|----------|
| 새 지출결의서 | 새 결의서 작성 |
| 지출결의서 목록 | 결의서 목록 |

---

## 오프라인 사용

### 오프라인 상태 표시

인터넷 연결이 끊어지면 화면 상단에 **노란색 배너** 표시

```
🔌 오프라인 상태입니다. 작성한 내용은 자동으로 저장됩니다.
```

### 오프라인에서 가능한 작업

| 가능 ✅ | 불가능 ❌ |
|---------|----------|
| 지출결의서 작성 | 결재 제출 |
| 임시저장 | 결재선 조회 |
| 첨부파일 추가 | 예산 현황 조회 |

---

## 백그라운드 동기화

### 자동 동기화

온라인 복귀 시 오프라인에서 저장한 데이터가 **자동으로 서버와 동기화**

### 동기화 상태 표시

| 상태 | 배너 색상 | 설명 |
|------|----------|------|
| 오프라인 | 🟡 노란색 | 인터넷 연결 없음 |
| 동기화 대기 | 🔵 파란색 | N개 항목 대기 중 |
| 동기화 완료 | 🟢 녹색 | 모든 데이터 동기화됨 |
| 동기화 실패 | 🔴 빨간색 | 오류 발생 |

---

## 문제 해결

### 설치 옵션이 안 나타남
- iOS: **Safari** 브라우저 사용 확인
- Android: **HTTPS** 연결 확인
- 이미 설치된 경우 옵션 안 나타남

### 오프라인에서 작동 안 함
- 해당 페이지를 온라인에서 한 번 방문 필요
- 브라우저 캐시 용량 확인

### 동기화가 안 됨
1. 인터넷 연결 확인
2. "지금 동기화" 버튼 클릭
3. 페이지 새로고침
