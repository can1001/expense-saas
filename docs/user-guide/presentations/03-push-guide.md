# 푸시 알림 가이드

**결재 알림을 실시간으로 받아보세요**

---

## 웹 푸시 알림이란?

### 브라우저를 통한 실시간 알림

앱을 설치하지 않아도 중요한 결재 알림을 받을 수 있습니다

### 장점

| 장점 | 설명 |
|------|------|
| 실시간 알림 | 결재 이벤트 발생 시 즉시 알림 |
| 백그라운드 알림 | 브라우저를 닫아도 알림 수신 |
| 다기기 지원 | PC, 모바일 모두 알림 가능 |

---

## 지원 브라우저

| 브라우저 | 지원 여부 | 비고 |
|----------|----------|------|
| Chrome | ✅ | 권장 |
| Edge | ✅ | |
| Firefox | ✅ | |
| Safari | ⚠️ | macOS 13+, iOS 16.4+ |

> Safari에서는 PWA 설치 후 알림을 받을 수 있습니다

---

## 알림 권한 설정

### 권한 요청 과정

1. 사이트 접속 시 알림 권한 요청 배너 표시
2. **"알림 받기"** 버튼 클릭
3. 브라우저 권한 요청 팝업에서 **"허용"** 선택

### 권한 상태

| 상태 | 설명 |
|------|------|
| 허용 (granted) | ✅ 알림 수신 가능 |
| 거부 (denied) | ❌ 알림 차단됨 |
| 기본값 (default) | ⏳ 아직 선택 안 함 |

---

## 알림 종류

### 결재 관련 알림

| 이벤트 | 수신자 | 알림 내용 |
|--------|--------|----------|
| 결재 제출 | 결재자 | 새로운 결재 요청 도착 |
| 결재 승인 | 신청자 | 결재가 승인됨 |
| 결재 반려 | 신청자 | 결재가 반려됨 + 사유 |
| 결재 회수 | 결재자 | 결재 요청 회수됨 |
| 지급 완료 | 신청자 | 지급 완료됨 |

### 알림 클릭
→ 해당 지출결의서 상세 페이지로 이동

---

## 알림 설정 변경

### 설정 페이지 접근
프로필 아이콘 → **"설정"** → **"알림 설정"** 탭

### 채널별 설정
| 설정 | 설명 |
|------|------|
| SMS 알림 | 문자 메시지 수신 여부 |
| 카카오 알림톡 | 카카오톡 알림 수신 여부 |
| 웹 푸시 알림 | 브라우저 알림 수신 여부 |

### 이벤트별 설정
각 이벤트(제출/승인/반려/지급완료)별로 개별 설정 가능

---

## 문제 해결

### 알림이 오지 않음

1. **브라우저 설정 확인**
   - 주소창 자물쇠 아이콘 → 사이트 설정 → 알림 허용

2. **운영체제 설정 확인**
   - Windows: 설정 → 알림 → 브라우저 앱 허용
   - macOS: 시스템 환경설정 → 알림 → 브라우저 앱 허용

3. **방해 금지 모드 확인**
   - 방해 금지 모드 해제

---

## 권한 거부 시 다시 허용하기

### Chrome
1. 주소창 왼쪽 **자물쇠 아이콘** 클릭
2. **"사이트 설정"** 선택
3. **"알림"**을 **"허용"**으로 변경

### Safari
1. Safari → 환경설정 → 웹사이트 → 알림
2. 해당 사이트 설정을 **"허용"**으로 변경
