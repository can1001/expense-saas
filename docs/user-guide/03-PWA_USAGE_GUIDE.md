## 1. PWA란?

### 개요

PWA(Progressive Web App)는 웹 기술로 만든 앱입니다. 웹사이트처럼 접근하면서도 네이티브 앱과 유사한 사용자 경험을 제공합니다.

### 장점

| 장점 | 설명 |
|------|------|
| 설치 가능 | 홈 화면에 아이콘으로 추가 |
| 오프라인 지원 | 인터넷 없이도 사용 가능 |
| 빠른 로딩 | 리소스 캐싱으로 빠른 시작 |
| 자동 업데이트 | 앱스토어 없이 자동 업데이트 |
| 저용량 | 네이티브 앱 대비 적은 용량 |

### PWA vs 네이티브 앱

| 항목 | PWA | 네이티브 앱 |
|------|-----|------------|
| 설치 | 브라우저에서 바로 | 앱스토어 다운로드 |
| 업데이트 | 자동 (새로고침) | 수동 (앱스토어) |
| 저장 공간 | 수 MB | 수십~수백 MB |
| 오프라인 | 캐시 기반 지원 | 완전 지원 |

## 2. 앱 설치 방법

### iOS (iPhone, iPad)

![iOS 앱 설치](screenshots/pwa-install-ios.png)

**설치 순서**:

1. **Safari** 브라우저로 사이트 접속
2. 하단의 **공유 버튼** (□↑) 탭
3. 스크롤하여 **"홈 화면에 추가"** 선택
4. 이름 확인 후 **"추가"** 탭

> **중요**: iOS에서는 **Safari**에서만 PWA 설치가 가능합니다. Chrome, Firefox 등에서는 설치 옵션이 나타나지 않습니다.

---

### Android

![Android 앱 설치](screenshots/pwa-install-android.png)

**Chrome에서 설치**:

1. Chrome으로 사이트 접속
2. 주소창 옆 **메뉴 (⋮)** 탭
3. **"앱 설치"** 또는 **"홈 화면에 추가"** 선택
4. **"설치"** 확인

**자동 설치 배너**:

사이트를 여러 번 방문하면 화면 하단에 설치 배너가 자동으로 표시됩니다. **"설치"** 버튼을 탭하면 바로 설치됩니다.

---

### 데스크톱 (PC, Mac)

**Chrome**:

1. 사이트 접속
2. 주소창 오른쪽의 **설치 아이콘** (⊕ 또는 모니터 아이콘) 클릭
3. **"설치"** 확인

**Edge**:

1. 사이트 접속
2. 주소창 오른쪽의 **앱 설치 아이콘** 클릭
3. **"설치"** 확인

---

### 설치 후 사용

**앱 실행**:
- 홈 화면(모바일) 또는 앱 목록(데스크톱)에서 **"지출결의서"** 아이콘 탭/클릭
- 브라우저 주소창 없이 전체 화면으로 실행됩니다

**바로가기 기능**:

앱 아이콘을 길게 누르면 바로가기 메뉴가 나타납니다:
- **새 지출결의서**: 바로 새 결의서 작성 화면으로 이동
- **지출결의서 목록**: 결의서 목록 화면으로 이동

## 3. 오프라인 사용

### 오프라인 상태 표시

![오프라인 배너](screenshots/offline-banner.png)

인터넷 연결이 끊어지면 화면 상단에 노란색 배너가 표시됩니다.

```
🔌 오프라인 상태입니다. 작성한 내용은 자동으로 저장됩니다.
```

### 오프라인에서 가능한 작업

| 가능 | 불가능 |
|------|--------|
| 지출결의서 작성 | 결재 제출 |
| 임시저장 | 결재선 조회 (네트워크 필요) |
| 첨부파일 추가 | 다른 사용자 정보 조회 |
| 이전에 본 페이지 확인 | 실시간 예산 현황 조회 |

### 데이터 저장 방식

오프라인에서 작성한 데이터는 브라우저의 **IndexedDB**에 저장됩니다.

- 별도 설정 없이 자동 저장
- 브라우저를 닫아도 데이터 유지
- 온라인 복귀 시 자동 동기화

### 오프라인 저장 용량

| 항목 | 제한 |
|------|------|
| 지출결의서 | 무제한 (브라우저 용량 한도 내) |
| 첨부파일 | 각 5MB 이하 |

## 4. 백그라운드 동기화

### 자동 동기화

![동기화 상태](screenshots/sync-status.png)

온라인 복귀 시 오프라인에서 저장한 데이터가 자동으로 서버와 동기화됩니다.

**동기화 과정**:

1. 오프라인에서 지출결의서 작성 및 저장
2. 온라인 복귀
3. 자동으로 서버에 데이터 전송
4. 동기화 완료 알림

### 동기화 상태 표시

| 상태 | 배너 색상 | 설명 |
|------|----------|------|
| 오프라인 | 노란색 | 인터넷 연결 없음 |
| 동기화 대기 | 파란색 | N개 항목 동기화 대기 중 |
| 동기화 중 | 파란색 (회전) | 동기화 진행 중 |
| 동기화 완료 | 녹색 | 모든 데이터 동기화됨 |
| 동기화 실패 | 빨간색 | 오류 발생, 재시도 필요 |

### 수동 동기화

자동 동기화가 작동하지 않는 경우:

1. 배너의 **"지금 동기화"** 버튼 클릭
2. 또는 페이지 새로고침

### 브라우저 지원

| 브라우저 | Background Sync | 대체 방식 |
|----------|----------------|----------|
| Chrome | O | - |
| Edge | O | - |
| Firefox | X | 온라인 이벤트 기반 |
| Safari | X | 온라인 이벤트 기반 |

> **참고**: Background Sync를 지원하지 않는 브라우저에서도 온라인 복귀 시 동기화가 진행됩니다.

## 5. 앱 삭제 방법

### iOS

1. 홈 화면에서 앱 아이콘 길게 누르기
2. **"북마크 삭제"** 또는 **"삭제"** 선택

### Android

1. 홈 화면에서 앱 아이콘 길게 누르기
2. **"제거"** 또는 **"삭제"** 선택

또는:
1. 설정 > 앱 > 지출결의서
2. **"제거"** 선택

### 데스크톱

1. 앱 실행 후 메뉴(⋮) 클릭
2. **"지출결의서 제거"** 선택

## 6. 문제 해결

### 설치 옵션이 나타나지 않음

**확인 사항**:

- **iOS**: Safari 브라우저를 사용하고 있는지 확인
- **Android/Chrome**: HTTPS 연결인지 확인
- 이미 설치된 경우 설치 옵션이 나타나지 않음

**해결 방법**:

1. 브라우저 캐시 삭제
2. 다른 브라우저에서 시도
3. 사이트 재접속

---

### 오프라인에서 작동하지 않음

**원인**:

- 해당 페이지를 온라인에서 한 번도 방문하지 않음
- 브라우저 캐시가 가득 참
- Service Worker가 설치되지 않음

**해결 방법**:

1. 온라인 상태에서 사용할 페이지를 한 번 방문
2. 브라우저 캐시 용량 확인
3. 브라우저 설정에서 Service Worker 활성화 확인

---

### 동기화가 되지 않음

**확인 사항**:

1. 인터넷 연결 상태 확인
2. 브라우저가 최신 버전인지 확인
3. Service Worker 상태 확인

**해결 방법**:

1. 페이지 새로고침
2. "지금 동기화" 버튼 클릭
3. 브라우저 재시작
4. 앱 재설치

**Service Worker 확인 방법** (개발자용):

1. 개발자 도구 열기 (F12)
2. Application 탭 선택
3. Service Workers 메뉴 확인
4. Status가 "activated and is running"인지 확인

---

### 앱이 업데이트되지 않음

**원인**:

- Service Worker 캐시가 갱신되지 않음
- 앱이 백그라운드에서 계속 실행 중

**해결 방법**:

1. 앱을 완전히 종료 (멀티태스킹에서 제거)
2. 앱 다시 실행
3. 문제가 지속되면 앱 삭제 후 재설치

---

### 저장한 데이터가 사라짐

**원인**:

- 브라우저 캐시/데이터 삭제됨
- 시크릿/프라이빗 모드 사용

**예방 방법**:

- 시크릿 모드에서는 사용하지 않기
- 브라우저 캐시 삭제 시 "쿠키 및 사이트 데이터" 제외
- 중요한 데이터는 온라인 상태에서 저장

---

### 자주 묻는 질문

**Q: PWA와 웹사이트의 차이점은?**

A: PWA는 홈 화면에 설치하여 앱처럼 사용할 수 있고, 오프라인에서도 작동합니다. 웹사이트는 항상 브라우저를 통해 접속해야 합니다.

**Q: 데이터 사용량이 많이 드나요?**

A: 최초 설치 시 필요한 리소스를 캐싱하므로, 이후에는 데이터 사용량이 크게 줄어듭니다.

**Q: 여러 기기에서 사용할 수 있나요?**

A: 네, 같은 계정으로 로그인하면 여러 기기에서 사용할 수 있습니다. 각 기기에서 앱을 설치하세요.

**Q: 알림은 어떻게 받나요?**

A: 웹 푸시 알림 설정이 필요합니다. [푸시 알림 가이드](./04-PUSH_NOTIFICATION_GUIDE.md)를 참고하세요.

**Q: 앱 용량은 얼마나 되나요?**

A: 약 5-10MB 정도로, 네이티브 앱 대비 매우 작습니다.
