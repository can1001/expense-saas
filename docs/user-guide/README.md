## 빠른 시작

### 지출결의서 작성

1. 상단 메뉴 또는 "+" 버튼으로 새 지출결의서 시작
2. 예산 정보 선택 (위원회 → 사역팀 → 항 → 목)
3. 세부 항목 입력 (세목, 적요, 단가, 수량)
4. 신청 정보 및 은행 정보 입력
5. 첨부파일 추가 (선택)
6. "저장" (임시저장) 또는 "제출" (결재요청)

### 앱 설치 (PWA)

| 플랫폼 | 방법 |
|--------|------|
| iOS | Safari > 공유 > "홈 화면에 추가" |
| Android | Chrome > 메뉴 > "앱 설치" |
| Desktop | Chrome/Edge > 주소창 설치 아이콘 클릭 |

### 알림 설정

1. 사이트 접속 시 알림 권한 허용
2. 프로필 > 설정 > 알림 설정에서 채널/이벤트 설정

## 주요 기능

| 기능 | 설명 |
|------|------|
| 지출결의서 작성 | 5단계 예산 계층, 다중 항목, 첨부파일 |
| 결재 시스템 | 3단계 결재 (담당자 → 회계 → 재정팀장) |
| 오프라인 저장 | 오프라인에서 작성, 온라인 복귀 시 동기화 |
| 푸시 알림 | 결재 이벤트 실시간 알림 |
| PWA | 앱 설치, 홈 화면 바로가기 |
