# 예산 계층별 담당자/예산금액 조회 화면 설계

> 작성일: 2026-01-06

## 1. 개요

### 1.1 요구사항
- 위원회 > 사역팀(부) > 예산(항) > 예산(목) > 예산(세목) 계층 구조로 예산 정보 조회
- 각 세목별 담당자, 예산금액 표시
- 기존 "세목별 담당자 관리" 페이지와의 차이점: **조직 중심 뷰** vs 예산항목 중심 뷰

### 1.2 기존 페이지와의 비교

| 구분 | 세목별 담당자 관리 (`/admin/budget-managers`) | 예산 현황 조회 (신규) |
|------|----------------------------------------------|----------------------|
| 계층 | 예산(항) > 예산(목) > 예산(세목) | 위원회 > 사역팀 > 예산(항) > 예산(목) > 예산(세목) |
| 목적 | 담당자/예산금액 **설정** | 담당자/예산금액 **조회** |
| 기능 | 수정 가능 (드롭다운, 입력) | 읽기 전용 |
| 관점 | 예산 항목 중심 | 조직 중심 |

## 2. 데이터 구조

### 2.1 관련 테이블

```
Committee (위원회)
    └── Department (사역팀)
            └── DepartmentBudgetDetail (사역팀-세목 연결)
                    └── BudgetDetail (세목)
                            └── BudgetSubcategory (목)
                                    └── BudgetCategory (항)

BudgetDetailYear (연도별 설정)
    ├── managerId → User (담당자)
    └── budgetAmount (예산금액)
```

### 2.2 쿼리 설계

조직 중심으로 조회하려면 `DepartmentBudgetDetail`을 기준으로 조인:

```typescript
// 위원회별 사역팀과 연결된 세목 조회
const data = await prisma.committee.findMany({
  where: { isActive: true },
  include: {
    departments: {
      where: { isActive: true },
      include: {
        budgetDetails: {
          include: {
            budgetDetail: {
              include: {
                subcategory: {
                  include: { category: true }
                },
                yearSettings: {
                  where: { year },
                  include: { manager: true }
                }
              }
            }
          }
        }
      }
    }
  }
});
```

## 4. 기능 상세

### 4.1 필터 기능

| 필터 | 설명 |
|------|------|
| 연도 선택 | 2024 ~ 2027년 (현재연도 기본) |
| 검색 | 위원회, 사역팀, 세목명, 담당자명 검색 |
| 위원회 필터 (선택) | 특정 위원회만 보기 |

### 4.2 표시 옵션

| 옵션 | 설명 |
|------|------|
| 전체 펼치기/접기 | 모든 위원회/사역팀 한번에 펼치기/접기 |
| 예산 없는 항목 숨기기 | `budgetAmount = 0` 항목 숨김 |
| 담당자 없는 항목 강조 | `managerId = null` 항목 빨간색 표시 |

### 4.3 통계 요약

페이지 상단에 요약 정보 표시:
- 총 위원회 수
- 총 사역팀 수
- 총 세목 수
- 총 예산 금액
- 담당자 미지정 세목 수 (경고)

## 9. 참고사항

### 9.1 데이터 관계

- `DepartmentBudgetDetail`: 사역팀과 세목의 N:M 관계 테이블
- 하나의 세목이 여러 사역팀에서 사용 가능
- 이 화면에서는 **사역팀 기준**으로 중복 표시될 수 있음

### 9.2 기존 페이지와의 차이

| 기존 (budget-managers) | 신규 (budget-view) |
|------------------------|-------------------|
| 세목 → 연결된 사역팀 표시 | 사역팀 → 사용 가능 세목 표시 |
| 수정 가능 | 조회 전용 |
| 예산 항목 중심 | 조직 중심 |

이 차이로 인해 **같은 데이터를 다른 관점**에서 볼 수 있음.
