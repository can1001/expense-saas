### 5. PrintFooter에 출납(인) 행 추가

- 지급일자 + 출납 담당자명 + (인) + 서명 이미지

```
변경 후 (4행, 지급완료 시):
┌──────────┬──────────────────────────────┐
│ 청구일자 │ 청구인 + (인) + 서명          │
├──────────┼──────────────────────────────┤
│ 입금은행 │ 계좌번호                      │
├──────────┼──────────────────────────────┤
│ 예금주   │ 청구팀(부)                    │
├──────────┼──────────────────────────────┤
│ 지급일자 │ 출납 + (인) + 서명            │  ← 신규
└──────────┴──────────────────────────────┘
```
