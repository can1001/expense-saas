# 결재선 설계 문서

## 개요

지출결의서 시스템의 결재선은 예산 마스터(BudgetMaster)의 위원회/부서 구조를 기반으로 자동 생성됩니다.

---

|--------|------|----------|
| 청연정혜종 | 재정팀장 | 최종 결재 (2차 또는 3차) |
| 청연윤운문 | 회계 | 2차 결재 |
| 청연신창국 | 팀장 (예배위원회) | 1차 결재 |

## 3. 결재 프로세스

### 기본 3단계 결재
```
작성자 → 1차 팀장 → 2차 회계 → 3차 재정팀장 (최종)
```

### 자기결재 방지 (동적 단계)
작성자가 결재선에 포함된 경우 해당 단계를 자동으로 건너뜁니다.

| 작성자 | 결재선 |
|--------|--------|
| 일반 사용자 | 1차 팀장 → 2차 회계 → 3차 재정팀장 (3단계) |
| 청연신창국 (팀장) | 1차 회계 → 2차 재정팀장 (2단계) |
| 청연김흥래 (팀장) | 1차 회계 → 2차 재정팀장 (2단계) |
| 청연윤운문 (회계) | 1차 팀장 → 2차 재정팀장 (2단계) |
| 청연정혜종 (재정팀장) | 담임목사 승인 필요 (미구현) |

---

## 4. 상태 전이

```
DRAFT (작성중)
   ↓ [제출]
PENDING (1차 결재 대기)
   ↓ [1차 승인]
APPROVED_STEP_1 (2차 결재 대기)
   ↓ [2차 승인]
APPROVED_STEP_2 (3차 결재 대기) - 3단계인 경우
   ↓ [최종 승인]
APPROVED_FINAL (최종 승인)

※ 어느 단계에서든 반려 시 → REJECTED
※ 작성자가 회수 시 → WITHDRAWN → 재제출 가능
```

## 6. 특수 케이스

### 재정팀 작성자
- 재정팀 소속 작성자는 회계(청연윤운문)가 본인일 수 있음
- 이 경우 1차 팀장 → 2차 재정팀장 (2단계)

### 재정팀장 작성자
- 청연정혜종이 작성한 경우 최종 결재자가 없음
- 담임목사 결재선 추가 필요 (현재 미구현)
