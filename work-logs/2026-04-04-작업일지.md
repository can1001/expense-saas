### 4. 팀장 겸직 지원 및 데이터 통합

**커밋:** `a2c51cf` feat: 팀장 겸직 지원 및 departments 페이지 UserYearRole 연동

#### 4.1 팀장 데이터 불일치 문제

**문제:**
- `/admin/departments`: `Department.leaderId` 사용 (레거시)
- `/admin/manager-exceptions`: `UserYearRole` 사용 (연도별)
- 두 페이지에서 서로 다른 팀장 표시

**해결:**
- departments 페이지에서 `UserYearRole` 조회하여 팀장 표시
- 팀장 지정 시 `POST /api/users/year-roles` 호출

#### 4.2 팀장 겸직 지원 (스키마 변경)

**문제:**
- `@@unique([userId, year])` 제약으로 연도별 1개 역할만 가능
- 예: 윤운문 → 재정팀장 + 행정비 팀장 불가

**해결:**
```prisma
// 변경 전
@@unique([userId, year])

// 변경 후
@@unique([userId, year, department])
```

**수정 파일:**
- `prisma/schema.prisma` - 유니크 제약 변경
- `lib/services/user-service.ts` - setYearRole 함수 수정
- `prisma/seed.ts` - 새 제약 적용
- `lib/services/__tests__/user-service.test.ts` - 테스트 업데이트
- `app/api/users/year-roles/route.ts` - 에러 로깅 개선
