## 완료된 작업

### 1. Android PWA 하단 네비게이션 바 가림 문제 수정

**커밋:** `72c22b8` fix: Android PWA에서 하단 시스템 네비게이션 바 가림 문제 수정

**문제:**
- Android PWA 환경에서 MobileDrawer의 로그아웃 버튼이 시스템 네비게이션 바에 60% 가려짐
- `env(safe-area-inset-bottom)`은 iOS에서만 작동하고 Android에서는 0으로 계산됨

**구현 내용:**

#### 1.1 CSS 변수 기반 safe-area 시스템 도입
- `--safe-area-bottom`: iOS safe-area 값 (Android fallback 포함)
- `--android-nav-height`: Android PWA용 동적 패딩 (JS에서 설정)
- `--bottom-safe-area`: 최종 하단 여백 계산 (두 값 중 큰 값)

#### 1.2 useSafeArea 훅 생성
- Android 기기 및 PWA standalone 모드 감지
- 시스템 네비게이션 바 높이 추정 (48dp/32dp/16dp)
- CSS 변수 `--android-nav-height` 동적 설정

#### 1.3 MobileDrawer 레이아웃 재구성
- `absolute bottom-0` 방식에서 flexbox 레이아웃으로 변경
- 드로어 컨테이너에 `flex flex-col` 적용
- 로그아웃 버튼에 safe-area 패딩 적용

#### 1.4 다른 하단 고정 요소들에 safe-area 적용
- ExpenseForm 모바일 하단 버튼
- MobileFilterPanel 하단 버튼 영역
- Toast 컴포넌트 위치
